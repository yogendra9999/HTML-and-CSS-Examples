# Coffee Shop Webpage Using HTML and CSS

## Description
This project is a simple Coffee Shop webpage created using HTML and CSS. It showcases a coffee shop with a menu, about section, contact information, and a clean, user-friendly design.

## Features
- Home page
- Coffee menu with prices
- About Us section
- Contact information
- Simple and responsive design

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
beans.html
style.css
README.md
```

## How to Run
1. Download or clone the project.
2. Open `beans.html` in your web browser.

## Output
- Displays a coffee shop webpage with a menu, shop information, and contact details.
