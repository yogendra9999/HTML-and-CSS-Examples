# Fashion Store Webpage Using HTML and CSS

## Description
This project is a responsive Fashion Store webpage developed using HTML and CSS. It showcases various fashion product categories, highlights upcoming festival sales, and provides information about the store and available payment options. The project demonstrates the fundamentals of webpage layout, styling, tables, and content organization using HTML5 and CSS3.

## Features
- Attractive homepage with a fashion store banner
- Product categories with images (T-Shirts, Sunglasses, Shirts, Shoes, Pants, and Perfumes)
- Promotional shopping banner
- Upcoming sales section with discounts and dates
- Informative content about the fashion store
- Festival sales information
- Payment options section
- Responsive and user-friendly interface
- Footer with copyright information

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
ex1.html
style.css
README.md
```

## How to Run
1. Download or clone the repository.
2. Open `ex1.html` in any modern web browser.

## Output
- Displays a responsive fashion store homepage.
- Shows product categories with images.
- Displays an attractive shopping banner.
- Presents upcoming festival sales in a table.
- Provides information about the fashion store and payment methods.
- Displays a footer with copyright details.

## Learning Outcomes
- Design responsive webpages using HTML and CSS.
- Create navigation-style layouts using images and text.
- Display tabular data using HTML tables.
- Organize webpage content into sections.
- Improve webpage presentation using CSS styling.
