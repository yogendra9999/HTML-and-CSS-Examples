# Hotel Webpage Using HTML and CSS

## Description
This project is a simple Hotel webpage created using HTML and CSS. It showcases different room types, their facilities, pricing, and a booking option with a modern and elegant design.

## Features
- Attractive hotel homepage
- Luxury, Business, and Resort room details
- Room facilities and pricing
- Book Now section
- Simple and responsive design

## Technologies Used
- HTML5
- CSS3

## Project Structure
```
hotel.html
style.css
README.md
```

## How to Run
1. Download or clone the project.
2. Open `hotel.html` in your web browser.

## Output
- Displays a hotel homepage with room categories, amenities, pricing, and booking information.
